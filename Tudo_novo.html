A tag H1 ou suas variantes, serve para adicionar uma escrita em seu site.

A tag P serve para adicionar em seu site um novo paragráfo.

A tag html representa a raiz do documento, junta todos os elementos

A tag Body é onde fica todo o conteúdo de texto do código, onde é adicionado ou modificado as frases.

A tag Meta é para definir as informações sobre os dados do documento.
